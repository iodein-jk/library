viewport

<meta name="viewport">の設定を変換します

デバイスの画面サイズの横幅を取得し、サイズに応じて[content]のwidthを書き換えます

viewportロード、リサイズ、デバイス回転のタイミングで発火し、viewportを更新します

変化の状態はブラウザの開発ツールを開くと確認可能です。

Getting Started

Download js

Live demo

Media Query
<meta name="viewport" content="">

開発ツールから<meta name="viewport" content="">タグを確認してください

import { viewport } from "./module/viewport.js"
let option = {
  mediaMaxWidth: 1200,
  minimum: 'width=device-width,initial-scale=1.0,viewport-fit=cover',
  width: 1500, 
}

viewport(option)

//以下表示確認用
window.addEventListener('DOMContentLoaded', function(){
  document.getElementById('media').textContent = viewport(option).media
  document.getElementById('content').textContent = viewport(option).content
})
Key Description Default
mediaMaxWidth メディアクエリで分岐するmax-widthを指定(px) 1000
minimum mediaMaxWidthの条件に入らない場合のviewportを設定 width=device-width,initial-scale=1.0
width mediaMaxWidthの条件に入った場合のviewportを設定 1280
