class Scroll

IntersectionObserverを使い、スクロールでコンテンツがエリア内に入ると、自動的に指定のclassを付与するイベントを設定します。
主にスクロールアニメーション等で使用。

Getting Started

Download js

Live demo

アニメーション確認用に、.animateと._animatedを別途スタイリングしています。

<div class="animate">
  <img src="http://placehold.jp/3d4070/ffffff/150x150.png" alt="">
</div>
import { class_Scroll } from "./module/class_Scroll.js"
window.addEventListener('DOMContentLoaded', function(){
  class_Scroll({
    select : ".animate",
    inClass: "_animated",
    forward: false,
    options: {
      root: null,
      rootMargin: "0px",
      threshold: 0
    }
  })
})

Parameter

Key Description Default Required
select 監視対象のDOMを指定 Required
inClass 監視対象がviewport内に入った場合に付与するclassを指定 ._animated
forward 監視対象がviewport外に出た場合、inClassを削除するか選択 true
options IntersectionObserverに挿入するオプション
root、rootMargin、thresholdを指定		 
options: {
  root: null,
  rootMargin: "-50% 0px",
  threshold: 0
}
