Cookie

ページを開いた時のCookieを保存し、ページを開いた回数を計測します。

関数を実行すると、返り値としてページの訪問回数が返されます

ブラウザのシークレットモードでCookieをリセットできます。

引数により、クッキーの保存日数を指定可能。（1=1日）

回目の訪問です

import { cookie } from "./module/cookie.js"
let counter = cookie(120)

window.addEventListener("DOMContentLoaded", function(){
  document.getElementById('cookieText').textContent = counter
})
