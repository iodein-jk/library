Template

vue.jsの機能を利用したHTMLのテンプレート構文を作成。vueの詳しい機能についてはhttps://ja.vuejs.org/guide/essentials/template-syntax.htmlを参照

本ページでは、cdnによるvue.jsの機能を利用したテンプレートの作成について解説します。

Live demo

Getting Started

以下のvueパッケージが必須

<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
<script src="https://unpkg.com/http-vue-loader"></script>

Create Code

読み込みたいページファイルに、コンポーネントを埋め込む箇所へidを挿入

<div id="template"></div>

Component

埋め込みたいテンプレートエリアを作成し、別名でファイルを保存

<template>
  <div class="template">
    <p>ここにvueのコンポーネントが挿入されています。</p>
  </div>
</template>

<script>
module.exports = {
  data() {
    return {
    };
  },
  /* == props == テンプレートタグの属性を登録 */
  methods: {
  },
  mounted() {
  }
};
</script>
JavScript

親ファイル（今回の場合はindex.html）へ戻り、<script>を作成。または外部ファイルからリンクし下記コードを挿入

el：内に最初に設定したid、components内に作成したコンポーネントファイルのパスを追加。

コンポーネント名（今回の場合はmy-template）は任意ですが、大文字小文字の区分けはありません。

vueを正しく記述したら、最初に設定したidのタグ内に、設定したコンポーネント名のタグを挿入します

var app = new Vue({
  el: '#template',

  /* == data == リアクティブなデータを登録 */
  data(){
    return{
    };
  },
  /* ==== components ==== */
  // -- 単一ファイルコンポーネントの登録
  components: {
    'my-template': httpVueLoader('./components/template.vue'),
  },
  mounted() {
  }
});
<div id="template">
  <my-template></my-template>
</div>
